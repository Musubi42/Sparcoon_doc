# Remote GIT | Github
## Requis 
    - Git : https://git-scm.com/
    - CMD et des compétences minimes

## Installation :

# Depuis un dossier de travail à ne pas supprimer

Ouvrir le CMD et se rendre sur le dossier de travail à l'aide de "cd".

Entrer :

```shell
git init

git remote add origin *Lien*
```
